# Страницы путеводителя

Страницы редактируются здесь: `content/<lang>/**/*.md` (сегодня — только `ru`).
Новый `.md` генератор подхватывает автоматически, регистрировать не нужно.

Полный workflow (трёхпроходная схема, frontmatter, директивы, `covers`, маршрут
исключений, разбор падений `docs:strict`) — в committed-гайде **`/docs/README.md`**
в корне репозитория.
